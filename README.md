# 🎬 こども映画カレンダー（二子玉川 109シネマズ）

子どもと映画に行くための、公開日・ムビチケ特典がひと目でわかるWebアプリ。
**子ども向け作品を優先表示**し、**二子玉川 109シネマズで見られる作品**だけを載せています。

👉 公開ページ: https://futosi.github.io/eiga/

## 機能
- 📋 リスト表示（子ども向け優先＋公開が近い順）／🗓 カレンダー表示（月ごと）
- 「子ども向けだけ」トグルで一般向け作品を非表示
- カードをタップで、あらすじ情報・ムビチケ特典・二子玉川の上映時間リンクを表示
- 「上映中」「あと◯日で公開」がひと目でわかる

## 自動更新の仕組み
- `.github/workflows/update.yml` が **毎週金曜 6:00(JST)** に `scripts/update.mjs` を実行
- 109シネマズの「上映中／公開予定」ページから、二子玉川（`?t=futakotamagawa`）で上映する作品を抽出し、
  各作品ページの構造化データから公開日などを取得して `data.js` を再生成 → 変更があれば自動コミット
- 手動更新: リポジトリの Actions タブ →「映画データ週次更新」→ Run workflow

## ムビチケ特典の追記
特典情報は109から自動取得できないため `overrides.json` に手動で登録します。
`match`（タイトル部分一致）と `mubi`（特典内容）を追記するだけ。`cat` で子ども向け分類の上書きも可能。

## ローカルで更新を試す
```
node scripts/update.mjs   # data.js / data.json を再生成
```
`index.html` をブラウザで開けば表示できます（`data.js` を読み込む方式なのでサーバー不要）。
